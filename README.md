# Проверка населенных пунктов

Streamlit приложение для проверки Excel файла с населенными пунктами.

## Что делает приложение

Приложение загружает Excel файл, дает выбрать лист, очищает написание населенных пунктов, ищет полные дубли и похожие варианты написания.

Логика дублей построена по актуальным рабочим колонкам:

- `Филиал новый`
- `РЭС новый`
- `Район`
- `Населенный пункт`

Колонка `Статус населенного пункта` не участвует в поиске дублей. Старые колонки `Филиал` и `РЭС` удаляются из итоговой выгрузки, если в файле есть новые колонки.

## Результат

После обработки формируется Excel файл с листами:

- `Исправленный файл` — исходные данные с очищенным НП и статусом проверки
- `Дубли` — строки, где найдены полные дубли
- `Подозрительные` — похожие варианты написания внутри одного филиала, РЭС и района
- `Сводка` — краткие итоги проверки

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```
