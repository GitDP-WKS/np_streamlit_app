# Парсинг населенных пунктов

Streamlit приложение для очистки и нормализации написания населенных пунктов в Excel файле.

## Что делает приложение

Приложение загружает Excel файл, дает выбрать нужный лист, показывает предпросмотр данных и предлагает выбрать столбец, который нужно обработать.

После этого пользователь выбирает, куда записать результат парсинга:

- в новую колонку
- в существующую колонку

Приложение очищает написание населенных пунктов, убирает лишние пробелы, выравнивает основные сокращения и приводит СНТ к формату `СНТ «Название»`.

Анализ дублей, поиск похожих написаний и отдельные листы дублей полностью удалены.

## Результат

После обработки формируется Excel файл с листами:

- `Исправленный файл` — исходные данные с результатом парсинга
- `Сводка` — краткие итоги обработки и выбранные столбцы

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```
